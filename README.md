##  Практические задания

###  Практическое задание 1 — Вёрстка страницы магазина
Вёрстка страницы магазина

###  Практическое задание 2 — CRUD-интерфейс (клиентский)
Страница управления товарами с операциями создания, чтения, обновления и удаления на чистом JavaScript (данные хранятся в памяти браузера).

###  Практическое задание 3 — Работа с DevTools и HTTP
Отчёты с результатами исследования HTTP-запросов и инструментов разработчика браузера.

###  Практическое задание 4 — REST API сервер
Express-сервер с CRUD-операциями для сущностей `Store` и `User`.

###  Практическое задание 5 — React-клиент
SPA-приложение на React + Vite, подключённое к REST API из задания 4.  
Реализован список продуктов с возможностью добавления, редактирования и удаления через модальное окно.

###  Практическое задание 7-8 — JWT авторизация
Express-сервер с полноценной JWT-авторизацией:
- Регистрация и вход пользователей
- Защищённые маршруты для CRUD товаров
- Swagger-документация API

###  Практическое задание 9-10 — Refresh-токены
React SPA с системой access/refresh-токенов:
- Автоматическое обновление токенов
- Axios Interceptors для обработки запросов
- Защита маршрутов на клиенте

### 👥 Практическое задание 11 — Роли и права доступа
Доработка приложения из задания 10:
- Система ролей (админ, пользователь, продавец)
- Разграничение прав доступа на сервере и клиенте
- Административная панель

##  Быстрый старт (на примере практики 11)

```bash
# Клонировать репозиторий
git clone <url-репозитория>
cd front-and-back

# Запуск бэкенда
cd практика11/backend
npm install
node app.js
# Сервер доступен на http://localhost:3001

# Запуск фронтенда (в новом терминале)
cd практика11/frontend
npm install
npm start
# Клиент доступен на http://localhost:3000
```
